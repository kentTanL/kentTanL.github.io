### 目录结构

```html
> /
  > _includes                 公有的的引入文件夹
  > _layouts                  共有的样式/布局文件夹
  > _post                     提交的文章目录
  > _sass                     
  > assets/css
  > css
  > fonts
  > images
  > img
  > js
  > less
  > script
  - _config.yml               公有的配置文件
  - baidu_sitemap.xml         百度收录 sitemap 文件
  - baidu_verify_ZMQJyf9ZOj.html  百度收录验证文件
  - feed.xml                      RSS 生成文件
  - google14d9104e5923de58.html   谷歌收录验证文件
  - index.html                    首页
```

### 本地启动

- Github 窗口，输入`bundle exec jekyll serve`
